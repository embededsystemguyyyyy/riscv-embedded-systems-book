# RISC-V Embedded Systems – Companion Repository

This repository supports hands-on learning in **RISC-V embedded systems**, providing practical examples, board support, and tooling for engineers, students, and hobbyists working close to the hardware.

It serves as the **official companion repository** to the book *Open Hardware Revolution: Practical Embedded Design*.

---

## About This Repository

This project is designed to bridge theory and practice in modern **embedded systems development using RISC-V**.  
It focuses on bare-metal programming, peripherals, interrupts, timers, and real-world board bring-up using open hardware platforms.

The repository is intentionally structured to grow with community contributions and real hardware experimentation.

---

## Who This Is For

This repository is intended for:

- Embedded systems learners exploring **RISC-V**
- Engineering students and educators
- Firmware developers transitioning from ARM or AVR
- Makers and hobbyists working with open hardware
- Readers of *Open Hardware Revolution: Practical Embedded Design*

A basic understanding of C and low-level programming is helpful but not required.

---

## What You’ll Find Here

- 📘 **Book companion materials**
- 🧪 Bare-metal RISC-V examples
- 🔌 Peripheral drivers and experiments
- ⏱️ Interrupt and timer demonstrations
- 🧰 Toolchain and debugging setup guides
- 📚 Clear documentation and FAQs

All examples are designed to be minimal, readable, and hardware-focused.

---

## Supported Hardware

The repository currently includes or plans support for:

- SiFive-based RISC-V boards
- GD32 RISC-V microcontrollers
- ESP32-C3 (RISC-V core)

Additional boards can be added by the community over time.

---

## How to Get Started

1. Clone the repository
2. Review the documentation in the `docs/` folder
3. Set up your toolchain using `toolchains/`
4. Start with the examples in `examples/baremetal/`
5. Explore board-specific notes in `boards/`

Each example is intentionally small and focused.
